# Archive — Earlier Continuous Optimization Experiments

This folder contains code and results from earlier experiments on **continuous
optimization** (CEC 2017/2022 benchmarks, NAS, Bayesian optimization variants).
These experiments preceded the discrete combinatorial framework described in the
PPSN 2026 paper and are preserved here for reference only.

**The current paper code lives in the top-level `src/`, `benchmarks/`, and
`results/` directories.**

## Contents

| Folder | Description |
|---|---|
| `src/` | Continuous ORC optimizers (CARS, L-SHADE, ORC-BO, NAS search, etc.) |
| `benchmarks/` | CEC 2017/2022 runners, NAS baselines, parameter sweeps |
| `results/` | CEC convergence CSVs, NAS results, diagnostic plots |
| `tests/` | Unit tests for the continuous optimization code |
| `scripts/` | Shell/PowerShell runner scripts |
